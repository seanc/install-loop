# install-loop [![NPM version](https://badge.fury.io/js/install-loop.svg)](https://npmjs.org/package/install-loop) [![Build Status](https://travis-ci.org/seanc/install-loop.svg?branch=master)](https://travis-ci.org/seanc/install-loop)

## Installation

```sh
$ npm install --save install-loop
```

## Usage

```js
var installLoop = require('install-loop');
installLoop();
```
